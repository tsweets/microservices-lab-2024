## Spring Init Short Cut
```

spring init \
--boot-version=3.3.3 \
--type=maven-project \
--java-version=17 \
--packaging=jar \
--name=product-service \
--package-name=org.beer30.microservices.core.product \
--groupId=org.beer30.microservices.core.product \
--dependencies=actuator,webflux \
--version=1.0.0-SNAPSHOT \
product-service

```
